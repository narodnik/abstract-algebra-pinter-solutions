Problem A — Examples of Operations

Which of the following are operations on the indicated set?

  1. a*b = √|ab| on Q

    This is not an operation on Q, because the square roots of many rationals are irrational. For instance, let a=1 and b=2. a*b = √|1*2| = √2 ∉ Q

  2. a*b = a ln b on R+

    This is an operation, because it uniquely maps every ordered pair (a,b) ∈ R+×R+ to R+.

  3. a*b is a root of x2 - a2b2 = 0 on R

    This is not an operation, as it does not uniquely map each ordered pair (a,b) to a single element. For instance, with a = b = 1, x = ±1.

  4. a*b = a - b on Z

    This is an operation, because it uniquely maps every ordered pair (a,b) ∈ Z×Z to Z.

  5. a*b = a - b, on N

    This is not an operation, because it will map many pairs to negative numbers, which are not in N.

  6. a*b = |a - b|, on N

    This is an operation, because it uniquely maps every ordered pair (a,b) ∈ N×N to N.
